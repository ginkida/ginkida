Geography degree, self-taught in a dorm room, then 11 years of shipping production code across taxi platforms, pharma e-commerce, fintech, and grocery delivery. Three roles in one person: a System Architect who thinks in business outcomes, a backend developer who writes the actual Go, Rust, PHP, and Python, and a DevOps engineer who keeps it alive at 3 AM.
